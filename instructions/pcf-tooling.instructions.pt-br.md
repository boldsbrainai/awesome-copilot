# Obter ferramentas para o Power Apps Component Framework

Use o Microsoft Power Platform CLI (interface de linha de comando) para criar, depurar e deploy componentes de código usando a estrutura de componentes do Power Apps. O Microsoft Power Platform CLI permite que os desenvolvedores criem componentes de código rapidamente. No futuro, ele será ampliado para incluir suporte a experiências adicionais de desenvolvimento e gerenciamento do ciclo de vida de aplicativos (ALM).

Mais informações: [Instalar o Microsoft Power Platform CLI](https://learn.microsoft.com/en-us/power-apps/developer/data-platform/powerapps-cli))

> **Importante**: Para deploy seu componente de código usando o Microsoft Power Platform CLI, você precisa ter um ambiente do Microsoft Dataverse com privilégios de administrador de sistema ou personalizador de sistema.
