# Suporte 

## Como registrar issues e obter ajuda

Este projeto usa issues do GitHub para rastrear bugs e solicitações de funcionalidades. Pesquise as issues existentes antes de registrar novas para evitar duplicidades. Para novas issues, registre seu bug ou solicitação de funcionalidade como uma nova issue.

Para obter ajuda ou fazer perguntas sobre o uso deste projeto, abra uma issue no GitHub.

Inclua uma das seguintes declarações no arquivo:

- **Awesome Copilot Prompts** está em desenvolvimento ativo e é mantido pela equipe do GitHub e da Microsoft **E PELA COMUNIDADE**. Faremos o possível para responder a solicitações de suporte, solicitações de funcionalidades e perguntas da comunidade em tempo hábil.
