---
name: my-pull-requests
description: 'Listar meus pull requests no repositório atual'
---

Pesquise no repositório atual (usando `#githubRepo` para obter as informações do repositório) e liste todos os pull requests encontrados (usando `#list_pull_requests`) que estejam atribuídos a mim.

Descreva o propósito e os detalhes de cada pull request.

Se um pull request estiver aguardando revisão, mencione isso na resposta.

Caso tenham ocorrido falhas na verificação do pull request, descreva-as e sugira possíveis soluções.

Se o Copilot não tiver realizado uma revisão, ofereça-se para solicitar uma usando a hashtag `#request_copilot_review`.
