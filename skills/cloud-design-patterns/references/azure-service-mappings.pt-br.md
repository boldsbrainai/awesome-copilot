# Mapeamentos de serviços do Azure

## Serviços comuns do Azure por padrão

- **Fila de mensagens**: Azure Service Bus, Fila de armazenamento do Azure, Event Hubs
- **Cache**: Azure Cache for Redis, Cache do Azure Front Door
- **API Gateway**: Azure API Management, Azure Application Gateway
- **Identidade**: Azure AD, Azure AD B2C
- **Configuração**: Azure App Configuration, Azure Key Vault
- **Armazenamento**: Azure Storage (Blob, Tabela, Fila), Azure Cosmos DB
- **Computação**: Azure Functions, Azure Container Apps, Azure Kubernetes Service
- **Streaming de eventos**: Azure Event Hubs, Azure Event Grid
- **CDN**: Azure CDN, Azure Front Door
