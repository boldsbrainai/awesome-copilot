# Núcleo Semântico para Python

Utilize esta referência quando o projeto de destino for escrito em Python.

## Fontes oficiais

- Repositório: <https://github.com/microsoft/semantic-kernel/tree/main/python>
- Exemplos: <https://github.com/microsoft/semantic-kernel/tree/main/python/samples>

## Orientações específicas para Python

- Utilize padrões assíncronos modernos em todas as operações do kernel.
- Adicione dicas de tipo e mantenha as APIs explícitas, mesmo em código dinâmico.
- Siga as práticas padrão de empacotamento e ambiente do Python para dependências e ferramentas.
- Verifique os exemplos mais recentes do Python antes de introduzir novas APIs, plugins ou padrões de orquestração.
