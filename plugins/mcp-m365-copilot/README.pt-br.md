# Plugin de Agentes M365 baseados em MCP

Coleção abrangente para criar agentes declarativos com integração Model Context Protocol para Microsoft 365 Copilot.

## Instalação

```bash
# Usando o Copilot CLI
copilot plugin install mcp-m365-copilot@awesome-copilot
```

## O que está incluído

### Comandos (comandos slash)

| Comando | Descrição |
|---------|-------------|
| `/mcp-m365-copilot:mcp-create-declarative-agent` | mcp-create-declarative-agent |
| `/mcp-m365-copilot:mcp-create-adaptive-cards` | mcp-create-adaptive-cards |
| `/mcp-m365-copilot:mcp-deploy-manage-agents` | mcp-deploy-manage-agents |

### Agentes

| Agente | Descrição |
|-------|-------------|
| `mcp-m365-agent-expert` | Assistente especialista para criar agentes declarativos baseados em MCP para Microsoft 365 Copilot com integração Model Context Protocol |

## Origem

Este plugin faz parte do [Awesome Copilot](https://github.com/github/awesome-copilot), uma coleção orientada pela comunidade de extensões do GitHub Copilot.
