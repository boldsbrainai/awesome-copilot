# Plugin de Desenvolvimento de Servidor MCP em Java

Kit completo para criar servidores Model Context Protocol em Java usando o MCP Java SDK oficial com reactive streams e integração com Spring Boot.

## Instalação

```bash
# Usando o Copilot CLI
copilot plugin install java-mcp-development@awesome-copilot
```

## O que está incluído

### Comandos (comandos slash)

| Comando | Descrição |
|---------|-------------|
| `/java-mcp-development:java-mcp-server-generator` | Gera um projeto completo de servidor Model Context Protocol em Java usando o MCP Java SDK oficial com reactive streams e integração opcional com Spring Boot. |

### Agentes

| Agente | Descrição |
|-------|-------------|
| `java-mcp-expert` | Assistência especializada para criar servidores Model Context Protocol em Java usando reactive streams, o MCP Java SDK oficial e integração com Spring Boot. |
