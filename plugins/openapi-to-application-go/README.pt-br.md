# OpenAPI to Application - Plugin Go

Gere aplicações Go prontas para produção a partir de especificações OpenAPI. Inclui scaffolding de projeto, geração de handlers, configuração de middleware e boas práticas de Go para APIs REST.

## Instalação

```bash
# Usando o Copilot CLI
copilot plugin install openapi-to-application-go@awesome-copilot
```

## O que está incluído

### Comandos (comandos slash)

| Comando | Descrição |
|---------|-------------|
| `/openapi-to-application-go:openapi-to-application-code` | Gera uma aplicação completa e pronta para produção a partir de uma especificação OpenAPI |

### Agentes

| Agente | Descrição |
|-------|-------------|
| `openapi-to-application` | Assistente especialista para gerar aplicações funcionais a partir de especificações OpenAPI |

## Origem

Este plugin faz parte do [Awesome Copilot](https://github.com/github/awesome-copilot), uma coleção orientada pela comunidade de extensões do GitHub Copilot.
