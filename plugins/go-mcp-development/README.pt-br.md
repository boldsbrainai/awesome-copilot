# Plugin de Desenvolvimento de Servidor MCP em Go

Kit completo para criar servidores Model Context Protocol (MCP) em Go usando o `github.com/modelcontextprotocol/go-sdk` oficial. Inclui instruções de boas práticas, um prompt para gerar servidores e um modo de chat especializado para orientação.

## Instalação

```bash
# Usando o Copilot CLI
copilot plugin install go-mcp-development@awesome-copilot
```

## O que está incluído

### Comandos (comandos slash)

| Comando | Descrição |
|---------|-------------|
| `/go-mcp-development:go-mcp-server-generator` | Gera um projeto completo de servidor MCP em Go com estrutura adequada, dependências e implementação usando o `github.com/modelcontextprotocol/go-sdk` oficial. |

### Agentes

| Agente | Descrição |
|-------|-------------|
| `go-mcp-expert` | Assistente especialista para criar servidores Model Context Protocol (MCP) em Go usando o SDK oficial. |

## Origem

Este plugin faz parte do [Awesome Copilot](https://github.com/github/awesome-copilot), uma coleção orientada pela comunidade de extensões do GitHub Copilot.
