# OpenAPI to Application - Plugin C# .NET

Gere aplicações .NET prontas para produção a partir de especificações OpenAPI. Inclui scaffolding de projeto ASP.NET Core, geração de controllers, integração com Entity Framework e boas práticas de C#.

## Instalação

```bash
# Usando o Copilot CLI
copilot plugin install openapi-to-application-csharp-dotnet@awesome-copilot
```

## O que está incluído

### Comandos (comandos slash)

| Comando | Descrição |
|---------|-------------|
| `/openapi-to-application-csharp-dotnet:openapi-to-application-code` | Gera uma aplicação completa e pronta para produção a partir de uma especificação OpenAPI |

### Agentes

| Agente | Descrição |
|-------|-------------|
| `openapi-to-application` | Assistente especialista para gerar aplicações funcionais a partir de especificações OpenAPI |
