# Plugin de Desenvolvimento Java

Coleção abrangente de prompts e instruções para desenvolvimento em Java, incluindo Spring Boot, Quarkus, testes, documentação e boas práticas.

## Instalação

```bash
# Usando o Copilot CLI
copilot plugin install java-development@awesome-copilot
```

## O que está incluído

### Comandos (comandos slash)

| Comando | Descrição |
|---------|-------------|
| `/java-development:java-docs` | Garante que tipos Java sejam documentados com comentários Javadoc e sigam boas práticas de documentação. |
| `/java-development:java-junit` | Obtém boas práticas para testes unitários com JUnit 5, incluindo testes orientados por dados |
| `/java-development:java-springboot` | Obtém boas práticas para desenvolver aplicações com Spring Boot. |
| `/java-development:create-spring-boot-java-project` | Cria o esqueleto de um projeto Java com Spring Boot |

## Origem

Este plugin faz parte do [Awesome Copilot](https://github.com/github/awesome-copilot), uma coleção orientada pela comunidade de extensões do GitHub Copilot.
