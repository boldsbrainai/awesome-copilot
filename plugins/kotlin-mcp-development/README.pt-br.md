# Plugin de Desenvolvimento de Servidor MCP em Kotlin

Kit completo para criar servidores Model Context Protocol (MCP) em Kotlin usando a biblioteca oficial `io.modelcontextprotocol:kotlin-sdk`. Inclui instruções de boas práticas, um prompt para gerar servidores e um modo de chat especializado para orientação.

## Instalação

```bash
# Usando o Copilot CLI
copilot plugin install kotlin-mcp-development@awesome-copilot
```

## O que está incluído

### Comandos (comandos slash)

| Comando | Descrição |
|---------|-------------|
| `/kotlin-mcp-development:kotlin-mcp-server-generator` | Gera um projeto completo de servidor MCP em Kotlin com estrutura adequada, dependências e implementação usando a biblioteca oficial `io.modelcontextprotocol:kotlin-sdk`. |

### Agentes

| Agente | Descrição |
|-------|-------------|
| `kotlin-mcp-expert` | Assistente especialista para criar servidores Model Context Protocol (MCP) em Kotlin usando o SDK oficial. |
