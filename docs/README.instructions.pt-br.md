# 📋 Instruções Personalizadas

Instruções específicas para equipe e projeto, com o objetivo de aprimorar o comportamento do GitHub Copilot para tecnologias e práticas de codificação específicas.
### Como Contribuir

Consulte o [CONTRIBUTING.md](../CONTRIBUTING.md#adding-instructions) para obter diretrizes sobre como contribuir com novas instruções, melhorar as existentes e compartilhar seus casos de uso.

### Como Usar as Instruções Personalizadas

**Para Instalar:**
- Clique no botão de instalação do **VS Code** ou do **VS Code Insiders** para a instrução que deseja utilizar
- Baixe o arquivo `*.instructions.md` e adicione-o manualmente à coleção de instruções do seu projeto

**Para Usar/Aplicar:**
- Copie estas instruções para o arquivo `.github/copilot-instructions.md` do seu workspace
- Crie arquivos `*.instructions.md` específicos para tarefas na pasta `.github/instructions/` do seu workspace (por exemplo, `.github/instructions/my-csharp-rules.instructions.md`)
- As instruções são aplicadas automaticamente ao comportamento do Copilot após a instalação no seu workspace
